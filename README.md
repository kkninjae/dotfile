# dotfile

My setup of Mac OS development environment.


## Installation

```sh
$ git clone https://github.com/kkninjae/dotfile.git ~/.dotfile

$ cd ~/.dotfile

$ sh INSTALL.sh
```


## Customization

Any file with suffix `rc.sh` in `custom` directory will be loaded into zsh when
starting any zsh process.
